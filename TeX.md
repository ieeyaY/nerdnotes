## Installation

### TeX Live

[TeX Live](https://tug.org/texlive/)
[TeX Live - ArchWiki](https://wiki.archlinux.org/title/TeX_Live)
[Tex Live/CJK - ArchWiki](https://wiki.archlinux.org/title/TeX_Live/CJK)

```bash
pacman -S texlive-most texlive-lang
```

- texlive-most: 基础安装
- texlive-lang: 语言包 (只有中文是必要的)
